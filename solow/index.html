20. The Solow-Swan Growth Model

In this lecture we review a famous model due to Robert Solow (1925--2023) and Trevor Swan (1918--1989).

The model is used to study growth over the long run.

Although the model is simple, it contains some interesting lessons.

We will use the following imports.

import matplotlib.pyplot as plt
import numpy as np

20.1The model¶

In a Solow--Swan economy, agents save a fixed fraction of their current incomes.

Savings sustain or increase the stock of capital.

Capital is combined with labor to produce output, which in turn is paid out to workers and owners of capital.

To keep things simple, we ignore population and productivity growth.

For each integer t≥0t \geq 0t≥0, output YtY_tYt​ in period ttt is given by Yt=F(Kt,Lt)Y_t = F(K_t, L_t)Yt​=F(Kt​,Lt​), where KtK_tKt​ is capital, LtL_tLt​ is labor and FFF is an aggregate production function.

The function FFF is assumed to be nonnegative and homogeneous of degree one, meaning that

F(λK,λL)=λF(K,L)for all λ≥0F(\lambda K, \lambda L) = \lambda F(K, L) \quad \text{for all } \lambda \geq 0F(λK,λL)=λF(K,L)for all λ≥0
(1)

Production functions with this property include

  • the Cobb-Douglas function F(K,L)=AKαL1−αF(K, L) = A K^{\alpha} L^{1-\alpha}F(K,L)=AKαL1−α with 0≤α≤10 \leq \alpha \leq 10≤α≤1.

  • the CES function F(K,L)={aKρ+bLρ}1/ρF(K, L) = \left\{ a K^\rho + b L^\rho \right\}^{1/\rho}F(K,L)={aKρ+bLρ}1/ρ with a,b,ρ>0a, b, \rho > 0a,b,ρ>0.

Here, α\alphaα is the output elasticity of capital and ρ\rhoρ is a parameter that determines the elasticity of substitution between capital and labor.

We assume a closed economy, so aggregate domestic investment equals aggregate domestic saving.

The saving rate is a constant sss satisfying 0≤s≤10 \leq s \leq 10≤s≤1, so that aggregate investment and saving both equal sYts Y_tsYt​.

Capital depreciates: without replenishing through investment, one unit of capital today becomes 1−δ1-\delta1−δ units tomorrow.

Thus,

Kt+1=sF(Kt,Lt)+(1−δ)KtK_{t+1} = s F(K_t, L_t) + (1 - \delta) K_tKt+1​=sF(Kt​,Lt​)+(1−δ)Kt​
(2)

Without population growth, LtL_tLt​ equals some constant LLL.

Setting kt:=Kt/Lk_t := K_t / Lkt​:=Kt​/L and using homogeneity of degree one now yields

kt+1=sF(Kt,L)L+(1−δ)KtL=sF(Kt,L)L+(1−δ)kt=sF(kt,1)+(1−δ)ktk_{t+1} = s \frac{F(K_t, L)}{L} + (1 - \delta) \frac{K_t}{L} = s \frac{F(K_t, L)}{L} + (1 - \delta) k_t = s F(k_t, 1) + (1 - \delta) k_tkt+1​=sLF(Kt​,L)​+(1−δ)LKt​​=sLF(Kt​,L)​+(1−δ)kt​=sF(kt​,1)+(1−δ)kt​
(3)

With f(k):=F(k,1)f(k) := F(k, 1)f(k):=F(k,1), the final expression for capital dynamics is

kt+1=g(kt) where g(k):=sf(k)+(1−δ)k k_{t+1} = g(k_t) \text{ where } g(k) := s f(k) + (1 - \delta) kkt+1​=g(kt​) where g(k):=sf(k)+(1−δ)k
(4)

Our aim is to learn about the evolution of ktk_tkt​ over time, given an exogenous initial capital stock k0k_0k0​.

20.2A graphical perspective¶

To understand the dynamics of the sequence (kt)t≥0(k_t)_{t \geq 0}(kt​)t≥0​ we use a 45-degree diagram.

To do so, we first need to specify the functional form for fff and assign values to the parameters.

We choose the Cobb--Douglas specification f(k)=Akαf(k) = A k^\alphaf(k)=Akα and set A=2.0A=2.0A=2.0, α=0.3\alpha=0.3α=0.3, s=0.3s=0.3s=0.3 and δ=0.4\delta=0.4δ=0.4.

The function ggg from The Solow-Swan Growth Model is then plotted, along with the 45-degree line.

Let’s define the constants.

A, s, alpha, delta = 2, 0.3, 0.3, 0.4
x0 = 0.25
xmin, xmax = 0, 3

Now, we define the function ggg.

def g(A, s, alpha, delta, k):
    return A * s * k**alpha + (1 - delta) * k

Let’s plot the 45-degree diagram of ggg.

def plot45(kstar=None):

Suppose, at some ktk_tkt​, the value g(kt)g(k_t)g(kt​) lies strictly above the 45-degree line.

Then we have kt+1=g(kt)>ktk_{t+1} = g(k_t) > k_tkt+1​=g(kt​)>kt​ and capital per worker rises.

If g(kt)<ktg(k_t) < k_tg(kt​)<kt​ then capital per worker falls.

If g(kt)=ktg(k_t) = k_tg(kt​)=kt​, then we are at a steady state and ktk_tkt​ remains constant.

(A steady state of the model is a fixed point of the mapping ggg.)

From the shape of the function ggg in the figure, we see that there is a unique steady state in (0,∞)(0, \infty)(0,∞).

It solves k=sAkα+(1−δ)kk = s Ak^{\alpha} + (1-\delta)kk=sAkα+(1−δ)k and hence is given by

k∗:=(sAδ)1/(1−α) k^* := \left( \frac{s A}{\delta} \right)^{1/(1 - \alpha)}k∗:=(δsA​)1/(1−α)
(5)

If initial capital is below k∗k^*k∗, then capital increases over time.

If initial capital is above this level, then the reverse is true.

Let’s plot the 45-degree diagram to show the k∗k^*k∗ in the plot.

kstar = ((s * A) / delta)**(1/(1 - alpha))
plot45(kstar)

From our graphical analysis, it appears that (kt)(k_t)(kt​) converges to k∗k^*k∗, regardless of initial capital k0k_0k0​.

This is a form of global stability.

The next figure shows three time paths for capital, from three distinct initial conditions, under the parameterization listed above.

At this parameterization, k∗≈1.78k^* \approx 1.78k∗≈1.78.

20.3Growth in continuous time¶

In this section, we investigate a continuous time version of the Solow--Swan growth model.

We will see how the smoothing provided by continuous time can simplify our analysis.

Recall that the discrete time dynamics for capital are given by kt+1=sf(kt)+(1−δ)ktk_{t+1} = s f(k_t) + (1 - \delta) k_tkt+1​=sf(kt​)+(1−δ)kt​.

A simple rearrangement gives the rate of change per unit of time:

Δkt=sf(kt)−δktwhereΔkt:=kt+1−kt\Delta k_t = s f(k_t) - \delta k_t \quad \text{where} \quad \Delta k_t := k_{t+1} - k_tΔkt​=sf(kt​)−δkt​whereΔkt​:=kt+1​−kt​
(6)

Taking the time step to zero gives the continuous time limit

kt′=sf(kt)−δktwithkt′:=ddtkt k'_t = s f(k_t) - \delta k_t \qquad \text{with} \qquad k'_t := \frac{d}{dt} k_tkt′​=sf(kt​)−δkt​withkt′​:=dtd​kt​
(7)

Our aim is to learn about the evolution of ktk_tkt​ over time, given an initial stock k0k_0k0​.

A steady state for (7) is a value k∗k^*k∗ at which capital is unchanging, meaning kt′=0k'_t = 0kt′​=0 or, equivalently, sf(k∗)=δk∗s f(k^*) = \delta k^*sf(k∗)=δk∗.

We assume f(k)=Akαf(k) = Ak^\alphaf(k)=Akα, so k∗k^*k∗ solves sAkα=δks A k^\alpha = \delta ksAkα=δk.

The solution is the same as the discrete time case---see (5).

The dynamics are represented in the next figure, maintaining the parameterization we used above.

Writing kt′=g(kt)k'_t = g(k_t)kt′​=g(kt​) with g(k)=sAkα−δkg(k) = s Ak^\alpha - \delta kg(k)=sAkα−δk, values of kkk with g(k)>0g(k) > 0g(k)>0 imply kt′>0k'_t > 0kt′​>0, so capital is increasing.

This shows global stability heuristically for a fixed parameterization, but how would we show the same thing formally for a continuum of plausible parameters?

In the discrete time case, a neat expression for ktk_tkt​ is hard to obtain.

In continuous time the process is easier: we can obtain a relatively simple expression for ktk_tkt​ that specifies the entire path.

The first step is to set xt:=kt1−αx_t := k_t^{1-\alpha}xt​:=kt1−α​, so that xt′=(1−α)kt−αkt′x'_t = (1-\alpha) k_t^{-\alpha} k'_txt′​=(1−α)kt−α​kt′​.

Substituting into kt′=sAktα−δktk'_t = sAk_t^\alpha - \delta k_tkt′​=sAktα​−δkt​ leads to the linear differential equation

xt′=(1−α)(sA−δxt) x'_t = (1-\alpha) (sA - \delta x_t)xt′​=(1−α)(sA−δxt​)
(8)

This equation, which is a linear ordinary differential equation, has the solution

xt=(k01−α−sAδ)e−δ(1−α)t+sAδx_t = \left( k_0^{1-\alpha} - \frac{sA}{\delta} \right) \mathrm{e}^{-\delta (1-\alpha) t} + \frac{sA}{\delta}xt​=(k01−α​−δsA​)e−δ(1−α)t+δsA​
(9)

(You can confirm that this function xtx_txt​ satisfies (8) by differentiating it with respect to ttt.)

Converting back to ktk_tkt​ yields

kt=[(k01−α−sAδ)e−δ(1−α)t+sAδ]1/(1−α) k_t = \left[ \left( k_0^{1-\alpha} - \frac{sA}{\delta} \right) \mathrm{e}^{-\delta (1-\alpha) t} + \frac{sA}{\delta} \right]^{1/(1-\alpha)}kt​=[(k01−α​−δsA​)e−δ(1−α)t+δsA​]1/(1−α)

Since δ>0\delta > 0δ>0 and α∈(0,1)\alpha \in (0, 1)α∈(0,1), we see immediately that kt→k∗k_t \to k^*kt​→k∗ as t→∞t \to \inftyt→∞ independent of k0k_0k0​.

Thus, global stability holds.

Stochastic Productivity

To bring the Solow--Swan model closer to data, we need to think about handling random fluctuations in aggregate quantities.

Among other things, this will eliminate the unrealistic prediction that per-capita output yt=Aktαy_t = A k^\alpha_tyt​=Aktα​ converges to a constant y∗:=A(k∗)αy^* := A (k^*)^\alphay∗:=A(k∗)α.

We shift to discrete time for the following discussion.

One approach is to replace constant productivity with some stochastic sequence (At)t≥1(A_t)_{t \geq 1}(At​)t≥1​.

Dynamics are now

kt+1=sAt+1f(kt)+(1−δ)kt k_{t+1} = s A_{t+1} f(k_t) + (1 - \delta) k_tkt+1​=sAt+1​f(kt​)+(1−δ)kt​

Now the long run convergence obtained in the deterministic case breaks down, since the system is hit with new shocks at each point in time.

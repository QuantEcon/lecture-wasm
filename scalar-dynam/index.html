19. Dynamics in One Dimension

19.1Overview¶

In economics many variables depend on their past values

For example, it seems reasonable to believe that inflation last year with affects inflation this year.

(Perhaps high inflation last year will lead people to demand higher wages to compensate, which will feed into higher prices this year.)

Letting πt\pi_tπt​ be inflation this year and πt−1\pi_{t-1}πt−1​ be inflation last year, we can write this relationship in a general form as

πt=f(πt−1)\pi_t = f(\pi_{t-1})πt​=f(πt−1​)
(1)

where fff is some function describing the relationship between the variables.

This equation is an example of one-dimensional discrete time dynamic system.

In this lecture we cover the foundations of one-dimensional discrete time dynamics.

(While most quantitative models have two or more state variables, the one-dimensional setting is a good place to learn foundations and understand key concepts.)

Let’s start with some standard imports:

19.2Some definitions¶

This section sets out the objects of interest and the kinds of properties we study.

19.2.1Composition of functions¶

For this lecture you should know the following.

If

  • ggg is a function from AAA to BBB and

  • fff is a function from BBB to CCC,

then the composition f∘gf \circ gf∘g of fff and ggg is defined by

(f∘g)(x)=f(g(x))(f \circ g)(x) = f(g(x))(f∘g)(x)=f(g(x))
(2)

For example, if

  • A=B=C=RA=B=C=\mathbb RA=B=C=R, the set of real numbers,

  • g(x)=x2g(x)=x^2g(x)=x2 and f(x)=xf(x)=\sqrt{x}f(x)=x​, then (f∘g)(x)=x2=∣x∣(f \circ g)(x) = \sqrt{x^2} = |x|(f∘g)(x)=x2​=∣x∣.

If fff is a function from AAA to itself, then f2f^2f2 is the composition of fff with itself.

For example, if A=(0,∞)A = (0, \infty)A=(0,∞), the set of positive numbers, and f(x)=xf(x) = \sqrt{x}f(x)=x​, then

f2(x)=x=x1/4f^2(x) = \sqrt{\sqrt{x}} = x^{1/4}f2(x)=x​​=x1/4
(3)

Similarly, if nnn is a positive integer, then fnf^nfn is nnn compositions of fff with itself.

In the example above, fn(x)=x1/(2n)f^n(x) = x^{1/(2^n)}fn(x)=x1/(2n).

19.2.2Dynamic systems¶

A (discrete time) dynamic system is a set SSS and a function ggg that sends set SSS back into to itself.

Examples of dynamic systems include

  • S=(0,1)S = (0, 1)S=(0,1) and g(x)=xg(x) = \sqrt{x}g(x)=x​

  • S=(0,1)S = (0, 1)S=(0,1) and g(x)=x2g(x) = x^2g(x)=x2

  • S=ZS = \mathbb ZS=Z (the integers) and g(x)=2xg(x) = 2 xg(x)=2x

On the other hand, if S=(−1,1)S = (-1, 1)S=(−1,1) and g(x)=x+1g(x) = x+1g(x)=x+1, then SSS and ggg do not form a dynamic system, since g(1)=2g(1) = 2g(1)=2.

  • ggg does not always send points in SSS back into SSS.

We care about dynamic systems because we can use them to study dynamics!

Given a dynamic system consisting of set SSS and function ggg, we can create a sequence {xt}\{x_t\}{xt​} of points in SSS by setting

xt+1=g(xt) with x0 given. x_{t+1} = g(x_t) \quad \text{ with } x_0 \text{ given}.xt+1​=g(xt​) with x0​ given.
(4)

This means that we choose some number x0x_0x0​ in SSS and then take

x0,x1=g(x0),x2=g(x1)=g(g(x0)),etc. x_0, \quad x_1 = g(x_0), \quad x_2 = g(x_1) = g(g(x_0)), \quad \text{etc.}x0​,x1​=g(x0​),x2​=g(x1​)=g(g(x0​)),etc.
(5)

This sequence {xt}\{x_t\}{xt​} is called the trajectory of x0x_0x0​ under ggg.

In this setting, SSS is called the state space and xtx_txt​ is called the state variable.

Recalling that gng^ngn is the nnn compositions of ggg with itself, we can write the trajectory more simply as

xt=gt(x0) for t=0,1,2,…x_t = g^t(x_0) \quad \text{ for } t = 0, 1, 2, \ldotsxt​=gt(x0​) for t=0,1,2,…
(6)

In all of what follows, we are going to assume that SSS is a subset of R\mathbb RR, the real numbers.

Equation (4) is sometimes called a first order difference equation

  • first order means dependence on only one lag (i.e., earlier states such as xt−1x_{t-1}xt−1​ do not enter into (4)).

19.2.3Example: a linear model¶

One simple example of a dynamic system is when S=RS=\mathbb RS=R and g(x)=ax+bg(x)=ax + bg(x)=ax+b, where a,ba, ba,b are constants (sometimes called ``parameters’').

This leads to the linear difference equation

xt+1=axt+b with x0 given.x_{t+1} = a x_t + b \quad \text{ with } x_0 \text{ given}.xt+1​=axt​+b with x0​ given.
(7)

The trajectory of x0x_0x0​ is

x0,ax0+b,a2x0+ab+b,etc.x_0, \quad a x_0 + b, \quad a^2 x_0 + a b + b, \quad \text{etc.}x0​,ax0​+b,a2x0​+ab+b,etc.
(8)

Continuing in this way, and using our knowledge of {doc}geometric series <geom_series>, we find that, for any t=0,1,2,…t = 0, 1, 2, \ldotst=0,1,2,…,

xt=atx0+b1−at1−a x_t = a^t x_0 + b \frac{1 - a^t}{1 - a}xt​=atx0​+b1−a1−at​
(9)

We have an exact expression for xtx_txt​ for all non-negative integer ttt and hence a full understanding of the dynamics.

Notice in particular that ∣a∣<1|a| < 1∣a∣<1, then, by (9), we have

xt→b1−a as t→∞x_t \to \frac{b}{1 - a} \text{ as } t \to \inftyxt​→1−ab​ as t→∞
(10)

regardless of x0x_0x0​.

This is an example of what is called global stability, a topic we return to below.

19.2.4Example: a nonlinear model¶

In the linear example above, we obtained an exact analytical expression for xtx_txt​ in terms of arbitrary non-negative integer ttt and x0x_0x0​.

This made analysis of dynamics very easy.

When models are nonlinear, however, the situation can be quite different.

For example, in a later lecture The Solow-Swan Growth Model, we will study the Solow-Swan growth model, which has dynamics

kt+1=sAktα+(1−δ)ktk_{t+1} = s A k_t^{\alpha} + (1 - \delta) k_tkt+1​=sAktα​+(1−δ)kt​
(11)

Here k=K/Lk=K/Lk=K/L is the per capita capital stock, sss is the saving rate, AAA is the total factor productivity, α\alphaα is the capital share, and δ\deltaδ is the depreciation rate.

All these parameter are positive and 0<α,δ<10 < \alpha, \delta < 10<α,δ<1.

If you try to iterate like we did in (8), you will find that the algebra gets messy quickly.

Analyzing the dynamics of this model requires a different method (see below).

19.3Stability¶

Consider a dynamic system consisting of set S⊂RS \subset \mathbb RS⊂R and ggg mapping SSS to SSS.

19.3.1Steady states¶

A steady state of this system is a point x∗x^*x∗ in SSS such that x∗=g(x∗)x^* = g(x^*)x∗=g(x∗).

In other words, x∗x^*x∗ is a fixed point of the function ggg in SSS.

For example, for the linear model xt+1=axt+bx_{t+1} = a x_t + bxt+1​=axt​+b, you can use the definition to check that

  • x∗:=b/(1−a)x^* := b/(1-a)x∗:=b/(1−a) is a steady state whenever a≠1a \not= 1a=1,

  • if a=1a = 1a=1 and b=0b=0b=0, then every x∈Rx \in \mathbb Rx∈R is a steady state,

  • if a=1a = 1a=1 and b≠0b \not= 0b=0, then the linear model has no steady state in R\mathbb RR.

19.3.2Global stability¶

A steady state x∗x^*x∗ of the dynamic system is called globally stable if, for all x0∈Sx_0 \in Sx0​∈S,

xt=gt(x0)→x∗ as t→∞x_t = g^t(x_0) \to x^* \text{ as } t \to \inftyxt​=gt(x0​)→x∗ as t→∞
(12)

For example, in the linear model xt+1=axt+bx_{t+1} = a x_t + bxt+1​=axt​+b with a≠1a \not= 1a=1, the steady state x∗x^*x∗

  • is globally stable if ∣a∣<1|a| < 1∣a∣<1 and

  • fails to be globally stable otherwise.

This follows directly from (9).

19.3.3Local stability¶

A steady state x∗x^*x∗ of the dynamic system is called locally stable if there exists an ϵ>0\epsilon > 0ϵ>0 such that

∣x0−x∗∣<ϵ    ⟹    xt=gt(x0)→x∗ as t→∞| x_0 - x^* | < \epsilon \; \implies \; x_t = g^t(x_0) \to x^* \text{ as } t \to \infty∣x0​−x∗∣<ϵ⟹xt​=gt(x0​)→x∗ as t→∞
(13)

Obviously every globally stable steady state is also locally stable.

Here is an example where the converse is not true.

Example 1

Consider the self-map ggg on R\mathbb{R}R defined by g(x)=x2g(x)=x^2g(x)=x2. The fixed point 1 is not stable.

For example, gt(x)→∞g^t (x)\to\inftygt(x)→∞ for any x>1x>1x>1.

However, 0 is locally stable, because −1<x<1-1<x<1−1<x<1 implies that gt(x)→0g^t (x)\to 0gt(x)→0 as t→∞t\to\inftyt→∞.

Since we have more than one fixed point, 0 is not globally stable.

19.4Graphical analysis¶

As we saw above, analyzing the dynamics for nonlinear models is nontrivial.

There is no single way to tackle all nonlinear models.

However, there is one technique for one-dimensional models that provides a great deal of intuition.

This is a graphical approach based on 45-degree diagrams.

Let’s look at an example: the Solow-Swan model with dynamics given in (11).

We begin with some plotting code that you can ignore at first reading.

The function of the code is to produce 45-degree diagrams and time series plots.

def subplots():
    "Custom subplots with axes throught the origin"
    fig, ax = plt.subplots()

    # Set the axes through the origin
    for spine in ['left', 'bottom']:
        ax.spines[spine].set_position('zero')
        ax.spines[spine].set_color('green')
    for spine in ['right', 'top']:
        ax.spines[spine].set_color('none')

    return fig, ax


def plot45(g, xmin, xmax, x0, num_arrows=6, var='x'):

    xgrid = np.linspace(xmin, xmax, 200)

    fig, ax = subplots()
    ax.set_xlim(xmin, xmax)
    ax.set_ylim(xmin, xmax)
    ax.set_xlabel(r'${}_t$'.format(var), fontsize=14)
    ax.set_ylabel(r'${}_{}$'.format(var, str('{t+1}')), fontsize=14)

    hw = (xmax - xmin) * 0.01
    hl = 2 * hw
    arrow_args = dict(fc="k", ec="k", head_width=hw,
            length_includes_head=True, lw=1,
            alpha=0.6, head_length=hl)

    ax.plot(xgrid, g(xgrid), 'b-', lw=2, alpha=0.6, label='g')
    ax.plot(xgrid, xgrid, 'k-', lw=1, alpha=0.7, label='45')

    x = x0
    xticks = [xmin]
    xtick_labels = [xmin]

    for i in range(num_arrows):
        if i == 0:
            ax.arrow(x, 0.0, 0.0, g(x), **arrow_args) # x, y, dx, dy
        else:
            ax.arrow(x, x, 0.0, g(x) - x, **arrow_args)
            ax.plot((x, x), (0, x), 'k', ls='dotted')

        ax.arrow(x, g(x), g(x) - x, 0, **arrow_args)
        xticks.append(x)
        xtick_labels.append(r'${}_{}$'.format(var, str(i)))

        x = g(x)
        xticks.append(x)
        xtick_labels.append(r'${}_{}$'.format(var, str(i+1)))
        ax.plot((x, x), (0, x), 'k', ls='dotted')

    xticks.append(xmax)
    xtick_labels.append(xmax)
    ax.set_xticks(xticks)
    ax.set_yticks(xticks)
    ax.set_xticklabels(xtick_labels)
    ax.set_yticklabels(xtick_labels)

    bbox = (0., 1.04, 1., .104)
    legend_args = {'bbox_to_anchor': bbox, 'loc': 'upper right'}

    ax.legend(ncol=2, frameon=False, **legend_args, fontsize=14)
    plt.show()

def ts_plot(g, xmin, xmax, x0, ts_length=6, var='x'):
    fig, ax = subplots()
    ax.set_ylim(xmin, xmax)
    ax.set_xlabel(r'$t$', fontsize=14)
    ax.set_ylabel(r'${}_t$'.format(var), fontsize=14)
    x = np.empty(ts_length)
    x[0] = x0
    for t in range(ts_length-1):
        x[t+1] = g(x[t])
    ax.plot(range(ts_length),
            x,
            'bo-',
            alpha=0.6,
            lw=2,
            label=r'${}_t$'.format(var))
    ax.legend(loc='best', fontsize=14)
    ax.set_xticks(range(ts_length))
    plt.show()

Let’s create a 45-degree diagram for the Solow-Swan model with a fixed set of parameters. Here’s the update function corresponding to the model.

def g(k, A = 2, s = 0.3, alpha = 0.3, delta = 0.4):
    return A * s * k**alpha + (1 - delta) * k

Here is the 45-degree plot.

xmin, xmax = 0, 4  # Suitable plotting region.

plot45(g, xmin, xmax, 0, num_arrows=0)

The plot shows the function ggg and the 45-degree line.

Think of ktk_tkt​ as a value on the horizontal axis.

To calculate kt+1k_{t+1}kt+1​, we can use the graph of ggg to see its value on the vertical axis.

Clearly,

  • If ggg lies above the 45-degree line at this point, then we have kt+1>ktk_{t+1} > k_tkt+1​>kt​.

  • If ggg lies below the 45-degree line at this point, then we have kt+1<ktk_{t+1} < k_tkt+1​<kt​.

  • If ggg hits the 45-degree line at this point, then we have kt+1=ktk_{t+1} = k_tkt+1​=kt​, so ktk_tkt​ is a steady state.

For the Solow-Swan model, there are two steady states when S=R+=[0,∞)S = \mathbb R_+ = [0, \infty)S=R+​=[0,∞).

  • the origin k=0k=0k=0

  • the unique positive number such that k=szkα+(1−δ)kk = s z k^{\alpha} + (1 - \delta) kk=szkα+(1−δ)k.

By using some algebra, we can show that in the second case, the steady state is

k∗=(szδ)1/(1−α)k^* = \left( \frac{sz}{\delta} \right)^{1/(1-\alpha)}k∗=(δsz​)1/(1−α)
(14)

19.4.2Complex dynamics¶

The Solow-Swan model is nonlinear but still generates very regular dynamics.
